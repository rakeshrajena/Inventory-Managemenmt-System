## Item Management Module

The **Item Management** page is the core module of the Inventory Management System, allowing administrators to create, update, and manage inventory items. This is made from opensource templates for learning purpose.

### Features

- Add new inventory items
- Update existing item details
- Delete items from inventory
- Upload product images
- Enable or disable item status
- Manage product pricing and stock information
- Apply item discounts
- Clear form inputs for new entries

### User Interface

The page consists of the following sections:

#### Navigation
- Item
- Purchase
- Vendor
- Sale
- Customer
- Search
- Reports

#### Item Details
The Item Details form includes:

| Field | Description |
|--------|-------------|
| Item Number | Unique identifier for the item |
| Product ID | Product or SKU reference |
| Item Name | Name of the inventory item |
| Status | Active or Inactive |
| Description | Product description |
| Discount % | Discount applied to the item |
| Quantity | Available quantity |
| Unit Price | Price per unit |
| Total Stock | Current stock level |

### Available Actions

- **Add Item** – Create a new inventory record.
- **Update** – Modify an existing item.
- **Delete** – Remove an item from the inventory.
- **Clear** – Reset all form fields.

### Purpose

This module provides a simple and efficient interface for maintaining inventory records. It enables administrators to manage product information, pricing, stock quantities, and availability from a centralized dashboard.
